# EngFleet

## Idea

The idea with this project is to build a fleet of Agents whose purpose is to make engineering management simpler and automated.

![alt text](<CleanShot 2025-10-13 at 13.53.32.png>)

## Dev

```bash
uv sync
source .venv/bin/activate
```

Run the ADK web UI:

```bash
cd agents
adk web
```
